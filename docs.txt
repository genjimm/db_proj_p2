Dashboard Page:
    - 输入图书id并点击查看图书 -> 跳转到图书细节页面
    - 输入图书id并点击删除图书 -> 显示删除成功或失败
    - 点击新增图书 -> 跳转到输入图书信息并添加页面
    - 点击更新图书 -> 跳转到输入图书信息并更新页面

    - 输入作者 ID 并点击查看作者 -> 跳转到作者细节页面
    - 输入作者 ID 并点击删除作者 -> 显示删除成功或失败
    - 点击新增作者 -> 跳转到输入作者信息并添加页面
    - 点击更新作者 -> 跳转到输入作者信息并更新页面

    - 输入用户id并点击查看租借 -> 跳转到显示租借记录页面

图书细节页面:
    - 点击添加副本 -> 输入状态信息(AVAILABLE OR UNAVAILABLE) 显示成功添加或失败
    - 点击查看所有副本 -> 跳转到显示所有副本页面
    - 输入作者信息并点击添加作者 -> 显示为该本图书添加作者成功或失败
    - 点击查看作者 -> 跳转到显示该图书所有作者页面
    - 点击租借图书副本 -> 跳转到租借页面，输入租借信息并显示成功或失败
    - 点击归还图书副本 -> 跳转到归还页面，输入归还图书信息(包含actual return date)，
      显示归还成功或失败


作者细节页面:
    - 点击查看作品 -> 跳转到显示该作者所有作品页面
